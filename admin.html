Admin Instructions

⚡ Pending News Queue

The news scanner runs daily at 8am and automatically extracts commitments and NIL deals from news headlines. Each item appears here for you to review. Click Approve to add it to the live site, or Reject to discard it. Always verify with the source link before approving.

➕ Adding Manually

Use the "+ Add" buttons to manually enter a commitment, NIL deal, or camp. The system automatically checks for duplicates based on athlete name and school before saving. If a duplicate is detected you'll be warned and can choose to override.

🔑 Adding a Service Key

The news scanner requires a Supabase SERVICE ROLE key (not the anon key) to write to the database. Add it in Netlify as SUPABASE_SERVICE_KEY. Get it from Supabase → Project Settings → API → service_role key.
